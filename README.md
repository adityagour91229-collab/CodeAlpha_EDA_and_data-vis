# 📊 CodeAlpha Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Data Analytics Internship – Task 2 (Exploratory Data Analysis)**.

The application allows users to upload a dataset and perform Exploratory Data Analysis (EDA) through an interactive interface. It helps identify trends, patterns, missing values, outliers, and relationships between variables using statistical summaries and visualizations.

---

## ✨ Features

- 📁 Upload CSV Dataset
- 👀 Data Preview
- 📋 Dataset Information
- 🔍 Missing Value Analysis
- 📊 Summary Statistics
- 📈 Correlation Matrix
- 📉 Histogram
- 📦 Box Plot
- 📍 Scatter Plot
- 📊 Bar Chart
- 🥧 Pie Chart
- 🔥 Heatmap
- 📌 Interactive Dashboard

---

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/adityagour91229-collab/CodeAlpha_EDA.git
```

### Install Dependencies

```bash
npm install
```

### Run the Project

```bash
npm run dev
```
